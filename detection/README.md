## Introduction
For 3d object detection on SiT Dataset, we use the MMDetection3D toolbox.

The performance of baselines are as follows.

|      Models     |  mAP  |
|:---------------:|:-----:|
|  PointPillars   | 0.319 |
| CenterPoint - P | 0.382 |
| CenterPoint - V | 0.382 |
